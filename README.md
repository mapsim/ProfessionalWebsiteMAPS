# Professional Website

Static personal website for a data engineer focused on Databricks and Azure.

## Files

- `index.html`: Main page structure and content
- `style.css`: Visual design and responsive layout
- `script.js`: Scroll reveal interactions

## Local Preview

Open the site directly in a browser:

- `file:///C:/Users/mauro/Documents/ProfessionalWebsiteMAPS/index.html`

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Push this folder to the `main` branch.
3. In the repository, open `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select `main` and `/ (root)`.
6. Save and wait for the site URL to be generated.

## Customize

Update the placeholder content in `index.html`:

- Name and contact details
- LinkedIn and GitHub links
- Real project achievements
- Certifications and credentials
